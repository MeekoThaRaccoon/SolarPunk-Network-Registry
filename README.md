This is the voluntary, shared registry for nodes in the SolarPunk Autonomous Network.

## How to Join
1.  **Fork the main template:** Use the [SolarPunk-Autonomous-Live](https://github.com/MeekoThaRaccoon/SolarPunk-Autonomous-Live) template.
2.  **Deploy your node:** Follow its README to enable GitHub Pages.
3.  **Add yourself here:** Fork this registry repo, edit `nodes.json`, and add your node's information in the same format as the existing entry.
4.  **Submit a Pull Request.** Once merged, all nodes in the network will discover you.

## The Protocol
All nodes follow the [SolarPunk Network Protocol](https://github.com/MeekoThaRaccoon/SolarPunk-Autonomous-Live/blob/main/SOLARPUNK_NETWORK_PROTOCOL.md).

## Node List
The live list of nodes is in [`nodes.json`](./nodes.json). Nodes use this to find each other.
